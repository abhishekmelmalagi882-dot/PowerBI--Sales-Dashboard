# 📊 Sales Performance Dashboard

## 📌 Project Overview

This Power BI dashboard provides an interactive analysis of sales performance using the Superstore Sales dataset. It helps users analyze sales trends, customer behavior, product performance, regional sales, and shipping performance.

---

## 🚀 Dashboard Features

- 📈 Total Sales
- 🛒 Total Orders
- 👥 Total Customers
- 💰 Average Sales
- 📅 Monthly Sales Trend
- 🌍 Sales by Region
- 🚚 Sales by Ship Mode
- 📦 Sales by Category
- 📊 Sales by Sub-Category
- 🏆 Top 10 Products
- ⭐ Top 10 Customers
- 🎯 Interactive Filters (State, Region, Category, Ship Mode, Year)

## 🎯 Key Insights

- Technology category generated the highest sales.
- West region recorded the highest sales.
- Standard Class was the most frequently used shipping mode.
- Monthly sales increased significantly toward the end of the year.
- Top customers contributed a large share of total revenue.
